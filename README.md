## About
This module introduces `matplotlib` and plotting different chart types within `pandas`.

A rundown of how to build different chart types with matplotlib, including heatmaps with seaborn, and an intro to matplotlib’s object-oriented interface.



## Learning Objectives

*You will be able to:*

- Apply appropriate chart types when exploring a data set.
- Explore patterns and relationships in data with bar charts and scatter plots.
- Create line charts to explore temporal trends in a data set.
- Understand the object-oriented way of working with matplotlib to fully control your visualizations.
- Generate heat maps in `seaborn`.
